Dados

Os dados usado nessa anlise sobre a despesa de deputados federais da legisLatura atual com a atividade parlamentar. A CEAP (cota para o exerccio da atividade parlamentar) uma cota nica mensal destinada a custear os gastos dos deputados exclusivamente vinculados ao exerccio da atividade parlamentar. Exemplos de despesa desse tipo so: passagens areas, telefonia, servios postais, manuteno de escritrios de apoio atividade parlamentar, assinatura de publicaes, fornecimento de alimentao ao parlamentar, hospedagem, outras despesas com locomoo, contemplando locao ou fretamento de aeronaves, veculos automotores e embarcaes, servios de txi, pedgio e estacionamento e passagens terrestres. Quais so os deputados que gastaram mais dinheiro da CEAP? Quais so os mais econmicos?

summarized_data = data %>% 
    filter(valorDocumento > 0) %>% 
    group_by(nomeParlamentar) %>% 
    summarise(gastos = sum(valorDocumento))

more_spenders <- summarized_data %>% 
  top_n(5, gastos) %>% 
  ggplot(aes(x = reorder(nomeParlamentar, gastos), y = gastos)) +
  guides(fill = guide_legend('Deputados')) +
  labs(x='Deputados', y='Gasto total') + 
  geom_col(width = 0.5) +
  coord_flip()

less_expenders <- summarized_data %>%
  arrange(gastos) %>%
  slice(1:5) %>%
  ggplot(aes(x = reorder(nomeParlamentar, gastos), y = gastos)) +
  guides(fill = guide_legend('Deputados')) +
  labs(x='Deputados', y='Gasto total') + 
  geom_col(width = 0.5) +
  coord_flip()

grid.arrange(more_spenders, less_expenders, nrow = 2)

rm_limit = data %>%
